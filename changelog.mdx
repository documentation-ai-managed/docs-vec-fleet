## Recent Updates

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added automated fuel efficiency reports with customizable thresholds. You now receive alerts when consumption exceeds `{target_mpg}` for specific vehicle classes.
- Introduced GPS geofencing for maintenance zones. Define virtual boundaries to trigger service reminders automatically.

## Improvements

- Enhanced reporting dashboard loads `>30%` faster with optimized data queries.
- Mobile app now supports offline expense logging, syncing when connectivity returns.

## Bug Fixes

- Fixed incorrect odometer readings in export CSV files.
- Resolved authentication timeout during long report generation sessions.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated tire pressure monitoring with real-time alerts via partnered telematics devices.
- Customizable driver scorecards based on fuel usage, safety events, and route efficiency.

## Improvements

- Reporting engine now supports scheduled PDF exports to your email.

## Bug Fixes

- Corrected vehicle assignment errors when bulk-importing from CSV.
- Patched security vulnerability in API token refresh endpoints (no user action required).
